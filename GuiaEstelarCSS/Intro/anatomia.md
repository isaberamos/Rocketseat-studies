# Seletores

Conecta um elemento HTML com o CSS. Exemplo: ID, classe etc

## Tipos

* Global selector `*`
* Element/Type selector `h1, h2, h3, p, div`
* ID selector `#box, #container`
* Class Selector `.red, .m-4`
* Attribute selector, pseudo-class, pseudo-element
